# Contributing Guidelines

Thank you for your interest in this project!


## Current Status
This project is currently developed by a small university team. External pull requests are **not accepted** at this time.


## How You Can Contribute
While direct code contributions are not open yet, you can still help us improve the project by:

- **Reporting bugs:** If you encounter any issues while using the project, please open a [bug report](../../issues/new?template=bug_report.md).
- **Suggesting features:** If you have an idea for an improvement, please use the [feature request template](../../issues/new?template=feature_request.md).
